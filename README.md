## Running Locally

```bash
# Clone the repo
git clone https://github.com/LoneWolf592/Mordern-Portfolio.git
cd Mordern-Portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open `http://localhost:5173` in your browser.
